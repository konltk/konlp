# KGUNER
KGUNER 한국어 개체명 인식기

KGUNER 한국어 개체명 인식기는 경기대학교 ICL(Intelligent Content Lab.)에서 만든 개체명 인식기 입니다.
Python 3.6.0 에서 구현하였습니다.

작성자 : 최성필, 김선우
작성일 : 2018.05.29
소속 : 경기대학교 문헌정보학과 ICL
사용방법 :
  1. kguner.py 의 KGUNER 객체를 선언
  2. KGUNER 객체의 .ner() 메소드 인자로 문자 리스트와 품사 리스트를 입력
  3. 예측된 개체명 결과 리스트 반환
 
자세한 사항은 kimsw@kyonggi.ac.kr 에 문의해 주세요.
